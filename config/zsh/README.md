## コマンド履歴検索

### 前方一致検索
[Ctrl + p], [Ctrl + n] で入力している文字列と前方一致するコマンド履歴を表示する

### インクリメンタルサーチ
[Ctrl + r] で、コマンド履歴を検索する画面を表示する

## ディレクトリ移動

- cd コマンドがなくても、ディレクトリ文字列を入力したらディレクトリを移動する。
- ディレクトリを移動した履歴を保持しており、***popd*** or ***pd*** で一つ前のディレクトリに戻る。

### 利用したディレクトリのインクリメンタルサーチ

***j*** で、利用したことのあるディレクトリ一覧を表示され、インクリメンタルサーチができる。  
ディレクトリは利用した順に並んでいる。

## ファイル検索

[Ctrl + f] カレントディレクトリ以下のファイルをインクリメンタルサーチする

[Ctrl + gf] カレントディレクトリ以下のgit管理されているファイルをインクリメンタルサーチする
